IMPORTANT: Replace these placeholder images with your own images

Required Images:
================

1. puzzle-image.jpg
   - This image will be used for the puzzle game
   - Recommended size: 600x600 pixels (square)
   - Format: JPG
   - Tip: Choose a meaningful photo that the recipient will recognize

2. background.jpg
   - This will be the background for the letter page
   - Recommended size: 1920x1080 pixels or larger
   - Format: JPG
   - Tip: Choose a romantic or beautiful scenic image

How to Add Images:
==================
1. Find or create your images
2. Rename them exactly as shown above
3. Place them in this 'assets' folder
4. Delete this README.txt file

The website will work without these images, but they make it much more personal and beautiful!
